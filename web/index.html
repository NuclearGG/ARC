Architecture

Five agents,
one city

Each agent handles a different layer — from reading sensors to clearing ambulance corridors.

01
Perception Agent
Reads queue lengths, pedestrian density, and signal states — builds a structured world model every tick.
02
Local Control Agent
Manages signal timing for a single intersection. One per zone — 9 total.
03
Global Coordinator
Resolves conflicts between local agents. Always targets the worst bottleneck first.
04
BACS — Compliance
Behavior-Aware Compliance System. Detects rule-breaking and adaptively enforces it.
05
Emergency Agent
Detects ambulances, estimates severity, and clears corridors in real time.
